WHAT IS CPU

CPU คืออะไร

เครื่อง CPU ถือว่าเป็นชุดของการประมวลผลที่ร่วมทุก Hardware สำคัญไว้ภายในหนึ่งเดียว โดยคำว่า CPU นั้นถูกย่อมาจาก Central Processing Unit มีหน้าที่ในการรวมทุกอุปกรณ์ที่เกี่ยวกับการประมวลผลข้อมูลภายในเครื่องคอมพิวเตอร์ แล้วคำนวณออกมาเพื่อให้สามารถใช้งานได้อย่างคล่องตัว โดยจะมีสัญญาณ Clock หรือการเคาะที่จะถูกเรียกว่าหน่วย การวัดความเร็วของการใช้งาน CPU โดยหน่วยความเร็วของซีพียูจะถูกเรียกว่า Herzt (Hz) และจะเป็นการวัดว่าความเร็วในการทำงานของเครื่อง CPU จะมีกี่ครั้งในรอบ 1 วินาที โดย CPU ของเครื่อง Pentium4 จะมีความเร็วขั้นต่ำจะอยู่ที่ 2.5 GHz หรือการทำงานที่สองพันห้าร้อยล้านครั้งต่อ 1 วินาที สัญญาณ Clock จึงจะมีความเร็วสูง ซึ่ง CPU ก็จะต้องทำงานเร็ว เพื่อทำให้การใช้งานเป็นไปอย่างคล่องตัวและทันใจผู้ใช้ แต่ราคาของเครื่อง CPU ที่ทรงประสิทธิภาพก็มักจะสูงตามไปด้วย
